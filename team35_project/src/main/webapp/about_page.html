The Sign Language Translator Tool

Sign language is something that has changed the way of those that are
hard of hearing communicate, as well as those who simply wish to learn the nature of
this language. Hand motions and facial expressions are key for conversation. However,
sometimes sign language can be a little tricky to learn and master. This tool was designed
to assist those learning the language, or may just need a refresher!

How To Use The Translator Tool

Using this tool is quite simple. See that small box to the left that reads
"Type Here"? Within that box, you may type any phrase you would like to
be translated in sign language. This feature ranges from something as small as
to a full phrase. Then...

The empty box to the right will display the translation into sign
language. To see it, all you have to do now is press the button that says
"Translate Phrase", and a set of images will appear in the box showing
the proper hand signs for your text.
